# Simple-RSA-Encryption
Program Java ini adalah alat enkripsi teks sederhana. Kode membaca input pengguna, menghapus simbol selain huruf, dan mengubahnya menjadi kapital. Setiap huruf kemudian disandikan memakai rumus matematika ($M^7 \mod 26$). Hasilnya adalah pesan acak yang berguna untuk mempelajari logika dasar keamanan data.
